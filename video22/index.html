Notes from the video

  1. px -> pixels is a basic unit. My 2k resolution has 2560x1600 = 40,96,000 pixels in monitor.
  2. In 2560x1600 -> monitor width is 2560 px and height is 1600 px
  3. When using universal selector to do margin and padding zero then make sure to give left margin for <ol> & <ul>
  4. margin:auto is used to center a block element.
  5. 1.5em -> means 1.5 times of the inherited size.
  6. 2rem -> means 2 times of the root element size.
  7. vmin -> 1% of the viewport's smaller dimension. ex. for laptops vmin is height & for mobile its width.
  8. vmax -> 1% of the viewport's larger dimension.
  9. Use min-height to make CSS BOX height Dynamic. Box will adapt accordingly to the content present.
  10. Use min-width to make CSS BOX width Dynamic. Box will adapt accordingly to the content present.
